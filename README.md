# (World AI)

ئەڤە پرۆژەکێ وێبێ یە بۆ نیشاندانا کومەکا باشترین و ناڤدارترین ئامرازێن زیرەکییا دەستکرد (AI) ل جیهانێ. ئارمانج ئەوە کو لیژنەیەکا بکارهێنەر-دۆست بهێتە دابینکرن بۆ گەهشتنا ب ساناهی ب ڤان ئامرازێن بهێز.

## دەربارەی پرۆژەی

ئەڤ مالپەرە هاتیە دروستکرن ژلایێ **شڤان جەمال** ڤە. ئارمانجا سەرەکی ئەوە کو باشترین و نویترین ئامرازێن زیرەکییا دەستکرد (AI) ل ئێک جهـ کۆمبکەت و پێشکێشی بکارهێنەران بکەت، داکو ب ساناهی مفای ژێ ببینن د کارێن خۆ یێن ڕۆژانە دا، ژ دروستکرنا وێنە و ڤیدیۆیان بگرە هەتا نڤیسین و کۆدینگێ.

## ئامرازێن AI یێن بەردەست
د ڤی پرۆژەی دا، مە باشترین ئامراز کۆمکرینە:
*   **Chatbots:** ChatGPT, Gemini, Claude, Copilot, Character.ai, Llama 3
*   **Image Generation:** Midjourney, Leonardo.ai
*   **Video:** Runway, Pika
*   **Music & Voice:** Suno AI, ElevenLabs
*   **Productivity & Search:** Perplexity, Gamma, Canva, Hugging Face

## تایبەتمەندیێن پرۆژەی
*   **دیزاینێ مۆدێرن:** بکارئینانا کارتان (Cards) ب شێوازێ 3D و Glassmorphism.
*   **ئەنیمەیشن:** بکارئینانا Lottie Files و Particle Effects بۆ پاشبنما.
*   **فلتەرکرن:** شیانا فلتەرکرنا ئامرازان ل دویڤ جۆرێ وان (Chatbot, Image, Video, htd).
*   **گەنجاندن (Responsiveness):** دیزاینەکا تایبەت بۆ کۆمپیوتەری (`desktop.html`) و مۆبایلێ (`mobile.html`).

## تەکنۆلۆژیایێن بکارئینایی

*   **HTML5**
*   **CSS3** (دگەل **Tailwind CSS**)
*   **JavaScript** (بۆ فلتەرکرن و ئەنیمەیشنان)
*   **Lottie** (بۆ ئەنیمەیشنێن ئایکۆنان)

## چاوا کارپێبکەم؟

باشترین ڕێک ئەوە فایلێ `index.html` ڤەکەی، کو ب شێوەیەکێ ئۆتۆماتیکی دێ تە ئاراستە کەت:
*   ئەگەر کۆمپیوتەر بیت: دچیتە `desktop.html`
*   ئەگەر مۆبایل بیت: دچیتە `mobile.html`
